# Meeting Optimization Environment

## 📌 Problem
Meetings are often long, unstructured, and unproductive, making it difficult to extract key insights and actionable tasks.

---

## 💡 Solution
This project simulates a **Meeting Optimization Environment** using an AI agent that:

- 📝 Summarizes meeting discussions  
- ✅ Extracts actionable tasks  
- ⏱️ Improves time efficiency  

---

## 🚀 Features

- 🔁 Step-based environment (like an agent system)
- 📊 Reward-based evaluation
- 🧠 AI-powered summarization (AI mode)
- ⚙️ Basic rule-based mode (for comparison)
- 📌 Task extraction from discussions

---

## 🛠️ Tech Stack

- Python  
- Hugging Face Transformers  
- PyTorch  
- Docker  

---

## ▶️ How to Run

### 🔹 Using Docker (Recommended)

```bash
docker build -t meeting-env .
docker run meeting-env

Note: The default mode is set to 'basic' for stable execution. AI mode can be enabled using --mode ai.
